World Assets — Assignment 2

For Assignment 2, I designed a simulation world that includes three benches with object assets placed on top of them. The objects were selected to match common COCO-style detection classes so they can be used for camera-based object detection during the robot simulation.

Notes

All objects were placed to support the object detection requirement of the assignment. The bench height was set to approximately 0.5m, and the objects were positioned at a visible height of around 0.5m–0.65m so they can be captured clearly from the robot camera view.

The screenshots were taken from the robot camera perspective during the simulation to show that the objects are visible and suitable for detection.
